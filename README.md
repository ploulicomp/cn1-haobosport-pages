# cn1-haobosport-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。页面内容涵盖不同主题与功能，不针对任何单一域名或具体网站。仓库以静态文件形式存储页面源码，便于浏览、分享和版本管理。

## 目录结构

```
cn1-haobosport-pages/
├── index.html          # 主入口页面（如有）
├── pages/              # 存放各独立 HTML 页面
│   ├── page-example1.html
│   ├── page-example2.html
│   └── ...
├── assets/             # 公共资源（如样式、脚本、图片）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

各页面文件命名清晰，资源文件统一存放在 `assets` 目录下，便于维护。

## 页面归档说明

- 每个 HTML 文件均为独立页面，可直接在浏览器中打开。
- 页面之间无强关联，可单独使用或引用。
- 部分页面可能依赖 `assets` 目录下的公共资源，请确保文件路径正确。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来修正页面错误、优化样式或增加新页面。
- 新增页面请放入 `pages` 目录，并保持命名规范。
- 公共资源更新后，请同步检查相关页面兼容性。
- 本仓库不涉及任何具体域名或商业用途，仅作为页面存档与分发使用。
